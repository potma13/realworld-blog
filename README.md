# Realworld Blog Platform

A blog platform built with React, featuring authentication, article management, and like functionality.

## 🚀 Live Demo

[View Live Site](https://realworldblog-eight.vercel.app/)

## 📋 Features

- User authentication (sign up / sign in)
- Article management (create, edit, delete)
- Like articles functionality
- Responsive design
- Protected routes
- Form validation with React Hook Form
- State management with Zustand

## 🛠️ Tech Stack

- React 18
- React Router v6
- React Hook Form
- Zustand
- Axios
- React Markdown
- Vercel (hosting)

## 📦 Installation

1. Clone the repository:
- git clone https://github.com/potma13/realworld-blog.git
- cd realworld-blog
2. Install dependencies:
- npm install
3. Run development server:
- npm run dev
